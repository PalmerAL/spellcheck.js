Spellcheck.js: a lightweight, pattern based spellchecker

What's different?

While most spellcheckers use a large dictionary of every word in English, spellcheck.js uses a small list of common words that it extends by making predictions based on common English spelling rules. This makes it much smaller than other spellcheckers (under 3k minified and gzipped).

How to use

notes:
- this only works correctly with single-word inputs - don't try to input multiple words.
- the getSuggestions() and getBestReplacement() assume the word is misspelled, so run checkSpelling() first.

checkSpelling(word)

Check to see if a word is spelled correctly. Returns true or false

getBestReplacement(word)

Get the best possible replacement for a misspelled word. Returns an object with {value, diff}, where value is the best possible replacement and diff is a number (can be infinite, but 0 is an exact match and 1 or higher is completely different).

getSuggestions(word)

Returns an array of possible suggestions for the given word
