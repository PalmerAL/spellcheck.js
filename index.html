Spellcheck.js: a javascript spellchecker

What is it?

Spellcheck.js is a javascript spellchecker that allows for checking if a word is spelled correctly, and finding the best replacement for a misspelled word. It uses Webster's 1913 dictionary, a Porter-stemmer, a list of common misspellings, and some regex. For finding replacements for a word, it uses the Sift4 string-distance algorithm.

How to use

Getting Started

Include the following anywhere on your page:

<script src="path/to/spellcheck.js"></script>
<script async defer src="path/to/dictionary.js"></script>

notes:
- this only works correctly with single-word inputs - don't try to input multiple words.
- the getBestReplacement() function assumes the word is misspelled, so run checkSpelling() first.

spellcheck.checkSpelling(word)

Check to see if a word is spelled correctly. Returns true or false

spellcheck.getBestReplacement(word)

Get the best possible replacement for a misspelled word. Returns a string with a replacement for the input.
